## Optional: private / work repository

Point the harness at a directory **outside** this repo (not committed):

```bash
# bash
export LINEAGESCOPE_PRIVATE_SCAN_PATH=/path/to/your/dbt-or-airflow-repo
python real-project-tests/run_validation.py
```

```powershell
# PowerShell
$env:LINEAGESCOPE_PRIVATE_SCAN_PATH = "C:\path\to\your\repo"
python real-project-tests\run_validation.py
```

Summaries use the label `private_workspace` and **redact** the path in JSON (`scan_root`: `(private path — redacted)`).
